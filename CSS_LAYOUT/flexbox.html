Kesimpulan skenario 1 adalah : flex digunakan untuk menampilkan sebuah div/list dengan cara horizontal.

Kesimpulan skenario 2 adalah : membuktikan bahwa flex digunakan untuk menampilkan div/list secara horizontal.

Kesimpulan skenario 3 adalah : elemen reverse digunakan untuk membalikkan urutan dalam flex.

Kesimpulan skenario 4 adalah : style flex-wrap digunakan untuk mengatur posisi teks ketika layar sudah tidak cukup.

Kesimpulan skenario 5 adalah : elemen flex-flow digunakan untuk menggabungkan flex-directiom (row) dan flex-wrap (wrap-reverse)

Kesimpulan skenario 6 adalah : elemen justify-content digunakan untuk mengatur posisi text.
